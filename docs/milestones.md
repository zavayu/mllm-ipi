# Milestones

## Milestone 1: Baseline image rendering
Generate one modified image with visible embedded text.

## Milestone 2: Mock end-to-end pipeline
Render image, query mock model, log result.

## Milestone 3: Local Qwen inference
Query Qwen2.5-VL on one generated image.

## Milestone 4: Config-driven experiments
Run many image/config combinations and save JSONL logs.

## Milestone 5: Evaluation
Compute ASR and generate plots.

## Milestone 6: Stealth metrics
Compute MSE, SSIM, and OCR detectability.

## Milestone 7: SAM single-mask placement
Use SAM to choose a region automatically.

## Milestone 8: Multi-mask placement
Split long prompts across multiple SAM masks.

## Milestone 9: UI demo
Build a Streamlit interface for demos.
